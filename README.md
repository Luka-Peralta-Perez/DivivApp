# DivivApp
DivivApp es una aplicación móvil nativa para Android enfocada en ayudar a los camareros con la división equitativa y transparente de pedidos en las mesas. La app permite a los camareros cargar los pedidos de cada persona de la mesa a su nombre desde el inicio del servicio, eliminando la fricción social al momento de pagar.
Integra el SDK de IA de Gemini para recomendar a cada comensal qué bebida acompaña mejor el plato elegido. Además, consume la API de Edamam para alertar sobre alérgenos y dietas (ej. Sin TACC, Vegano, Vegetariano), directamente en la interfaz.
